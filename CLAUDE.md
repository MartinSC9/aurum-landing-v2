# Project conventions

## Package manager: pnpm

## Stack
- React 18 + Vite 8
- react-router-dom (routing)
- react-icons (iconos)
- CSS puro (variables CSS, sin Tailwind)

## Design system
- Consultar `DESIGN.md` antes de crear o modificar cualquier componente visual.
- Paleta: negro + dorado. Dark theme only.
- Fuentes: Inter (body) + Playfair Display (titulos H1/H2).
- Nunca dejar placeholders grises — usar imagenes reales de Unsplash.

## Context
- Ver `../CONTEXTO.md` para info del cliente (Daniel Segura, Aurum Team).

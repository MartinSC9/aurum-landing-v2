# Project conventions

## Package manager: pnpm

## Stack
- React 19 + Vite 8
- react-router-dom (routing)
- react-icons (iconos)
- CSS puro (variables CSS, sin Tailwind)

## Design system
- Consultar `DESIGN.md` antes de crear o modificar cualquier componente visual.
- Paleta: deep blue-black (#0c0e14) + warm bronze (#b8926a). Dark theme only.
- Fuentes: Cormorant Garamond (display/titulos) + Manrope (body) + JetBrains Mono (labels).
- Estilo editorial asimetrico — nunca repetir el mismo layout entre secciones consecutivas.
- Nunca dejar placeholders grises — usar imagenes reales de Unsplash.

- Ver `../CONTEXTO.md` para info del cliente (Daniel Segura, Aurum Team).
